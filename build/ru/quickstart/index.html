Быстрый старт

  1. Для начала нужно создать Фид. Фид — это хранилище, в которое будут собираться подписки.
  2. В созданном фиде скачать Service Worker и загрузить его в корневую директорию вашего веб-сервера.
  3. Выбрать один из способов интеграции сбора подписок на странице созданного фида и начать собирать подписки
  4. Создать кампанию для рассылки уведомлений по собранной базе
