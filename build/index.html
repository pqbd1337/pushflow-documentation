Quick start

  1. First you need to create a Feed. A Feed is a repository to which subscriptions will be collected.
  2. In the created feed page download Service Worker and upload it to the root directory of your web-server.
  3. Choose one of the ways to integrate subscription collection on the page of the created feed and start collecting subscriptions.
  4. Create a campaign to send notifications to the collected push database
